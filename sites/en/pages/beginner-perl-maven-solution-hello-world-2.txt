<h2>Write a simple script that prints     Hello world</h2>

<code lang="perl">
print "hello world\n";
</code>

<code>
$ perl hello.pl
</code>

<code lang="perl">
use strict;
use warnings;

print "hello world\n";
</code>


<h2>Add comments to your code</h2>

<code lang="perl">
use strict;
use warnings;

print "hello world\n";  #comment

# some comment
</code>


<h2>Add user documentation to your code</h2>

<code lang="perl">
use strict;
use warnings;

print "hello world\n";  #comment

# some comment
=pod
=head1 Title
text
=cut
</code>

<code>
$ perl hello.pl
$ perldoc hello.pl
</code>

But that does not look really good and <hl>perldoc</hl> reported about some error too.

That's because POD requires empty rows around its tags:

<code lang="perl">
use strict;
use warnings;

print "hello world\n";  #comment

# some comment

=pod

=head1 Title

text

=cut
</code>


<code>
$ perldoc hello.pl

$ pod2html hello.pl > hello.html

$ lynx hello.html
</code>
