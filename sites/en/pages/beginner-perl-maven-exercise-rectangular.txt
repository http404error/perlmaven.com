Exercise: rectangular

Write a program that computes the area of a rectangular.

<ul>
  <li>Start by having two hard coded values.</li>
  <li>Multiply them and print the result.</li>
  <li>Then change the code to prompt the user for two numbers.</li>
  <li>Multiply them and print the result.</li>
  <li>Further modify the program to warn if one of the sides given was negative.</li>
</ul>
